# 自定义圆圈加载进度条

## 用法解释

xml文件下修改
```
这个是逆时针
app:direction="anticlockwise"
顺时针
app:direction="clockwise"
这个是外边圆的边宽
app:progress_width="50dp"
半径大小
app:outside_radius="100dp"
渐变的开始颜色
app:startColor="@color/colorAccent"
渐变的结束颜色
app:endColor="@color/colorPrimary"
```
